# Bomberman
> *A demo project inspired by the classic **Bomberman** game, developed in **C++** using the **SDL2** graphics library.*

  <img src="assets/demo.gif" alt="Gameplay" />
</p>

---

## Features
* **Random Map Generation:** A new map is generated every time the game starts, with wooden crates placed randomly across the map.

* **Cross-Shaped Explosion Mechanic:** Bombs automatically detonate after a countdown, creating cross-shaped explosions that destroy nearby wooden crates.

---

## Data Resources

* All graphics and technical resources used in this project are sourced from the open-source game asset platform [OpenGameArt.org](https://opengameart.org/).

---

## Controls

* **Move:** Press `W`, `A`, `S`, `D` to move.
* **Place Bomb:** Press `SPACE`.
